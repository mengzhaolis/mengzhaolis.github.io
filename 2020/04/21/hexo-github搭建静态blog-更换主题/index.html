    1、GitHub账号一个，创建自己的仓库

    4、安装hexo

    准备搭建

    如果你已经有GitHub的帐号了那么恭喜你，搭建个人blog对你来说易如反掌，如果你还没有GitHub账号那么不要着急只需要在浏览器中输入网址https://github.com 然后点击下图中标识的地方，提交以下资料即完成了注册

    现在我们假设你已经注册并且登录GitHub了，那么这个时候你需要创建一个仓库(点击网页右上角的加号)，来存储你自己blog的代码，wait wait当你在创建仓库的时候需要注意的是看下图

    创建好blog的仓库之后需要做两步检查，第一步检查本地是否安装git，直接在电脑桌面右键即可，第二步检查本地是否安装node，可以打开cmd然后键入 node -v，如果git 和node.js都已经安装，那么你就可以将刚刚创建的仓库克隆到本地，进行下一步操作

    桌面右键检查git出现如下即可使用

    cmd中键入node -v

    注意：如果你是首次使用git那么请注意我们需要将git的ssh添加到GitHub，这里请自行百度就不一 一赘述了

    如果你没有安装node.js那么你可以去 http://nodejs.cn/download/ 这里现在适合你点单的版本然后解压双击就可以安装了，需要注意的是最好给node配置一下全局变量，这样更加方便操作

    开始搭建

    首先执行 git clone xxxxx (你的blog地址)将刚刚创建的仓库克隆到本地

    然后cd xxx (你的仓库名称)进入到你的仓库里，执行完这些，下面就是激动人心的时刻了因为我们已经安装了node那我们直接使用npm来安装Hexo

    使用npm 安装 Hexo：

    npm install -g hexo-cli

    这是一个需要等待的过程，安装完成后我们初始化自己的博客

    hexo init blog

    注意：以下命令都是作用在刚刚初始化的blog中的所以，当你初始化完成blog的时候你需要cd blog

    我们按顺序执行以下命令

    hexo new test_my_site 含义：创建一个名为test_my_site.md的文件

    hexo g 含义：生成网页

    hexo s 含义：启动服务预览(在本地预览)
